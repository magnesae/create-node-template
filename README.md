# BSL-Template-App

## Get Started

```bash
git clone "https://github.com/magnesae/BSL-Template-App.git"
```

```bash
cd BSL-Template-App
```

```bash
npm install
```

```bash
npm run build
```

```bash
npm link
```

```bash
cd EXAMPLE/DIR && bsl-template-app
```
